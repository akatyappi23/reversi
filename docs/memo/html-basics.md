# HTML基礎メモ

## HTMLとは

**HyperText Markup Language** の略。Webページの「骨格」を作る言語。
**タグ** で文章を囲んで、意味や構造をブラウザに伝える。

---

## タグの基本構造

```html
<タグ名>内容</タグ名>
```

- `<タグ名>` → 開始タグ
- `</タグ名>` → 終了タグ（スラッシュがつく）
- 開始〜終了タグ全体を **要素（element）** と呼ぶ

### 属性（attribute）

```html
<a href="https://example.com" target="_blank">リンク</a>
```

- `href="..."` のように `属性名="値"` の形式でタグに情報を付け加える

---

## ページの基本構成

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8">
    <title>ページタイトル</title>
  </head>
  <body>
    ここに表示内容を書く
  </body>
</html>
```

| タグ | 役割 |
|------|------|
| `<!DOCTYPE html>` | HTML5であることをブラウザに宣言 |
| `<html>` | 全体を包む最外側の要素 |
| `<head>` | ブラウザ向け設定情報（画面に表示されない） |
| `<meta charset="UTF-8">` | 文字コード設定（日本語表示に必要） |
| `<title>` | ブラウザのタブに表示されるタイトル |
| `<body>` | 画面に表示される内容 |

## よく使うタグ

### 見出し `<h1>` 〜 `<h6>`

```html
<h1>最大の見出し</h1>
<h2>2番目</h2>
<h3>3番目</h3>
```

- 数字が大きいほど小さい見出し
- `<h1>` はページに1つだけが推奨

### 段落 `<p>`

```html
<p>段落のテキスト。前後に自動で余白が入る。</p>
```

### 改行 `<br>`

```html
1行目<br>
2行目
```

- 終了タグなしの**空要素**

### リンク `<a>`

```html
<a href="https://example.com" target="_blank">外部リンク</a>
```

| 属性 | 意味 |
|------|------|
| `href` | リンク先URL |
| `target="_blank"` | 新しいタブで開く |

### 画像 `<img>`

```html
<img src="photo.jpg" alt="写真の説明" width="300">
```

| 属性 | 意味 |
|------|------|
| `src` | 画像ファイルのパス |
| `alt` | 画像が表示できない時の代替テキスト |
| `width` / `height` | サイズ（ピクセル） |

### リスト

```html
<!-- 箇条書き -->
<ul>
  <li>りんご</li>
  <li>みかん</li>
</ul>

<!-- 番号付き -->
<ol>
  <li>手順1</li>
  <li>手順2</li>
</ol>
```

### テーブル（表）

```html
<table>
  <tr>
    <th>名前</th>
    <th>年齢</th>
  </tr>
  <tr>
    <td>田中</td>
    <td>25</td>
  </tr>
</table>
```

| タグ | 意味 |
|------|------|
| `<table>` | 表全体 |
| `<tr>` | 行（table row） |
| `<th>` | 見出しセル（太字・中央寄せ） |
| `<td>` | データセル（table data） |

### フォーム

```html
<form action="/submit" method="post">
  <label for="name">名前：</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="送信">
</form>
```

主な `<input>` の `type` 値：

| type | 内容 |
|------|------|
| `text` | テキスト入力 |
| `password` | パスワード（伏字） |
| `checkbox` | チェックボックス |
| `radio` | ラジオボタン |
| `submit` | 送信ボタン |
| `email` | メールアドレス入力 |

## グループ化タグ

### `<div>`（ブロック要素）

- 縦に積み重なる「見えない箱」
- それ自体に見た目はなく、CSSやJSと組み合わせて使う
- `class` や `id` 属性でスタイルを適用する

```html
<div class="card">
  <h2>カードタイトル</h2>
  <p>カードの内容</p>
</div>
```

### `<span>`（インライン要素）

- 文章の中の一部分を装飾するために使う

```html
<p>この<span style="color: red;">単語だけ</span>赤くしたい</p>
```

### div vs span

| タグ | 種類 | 用途 |
|------|------|------|
| `<div>` | ブロック要素 | 大きなブロック・セクションをまとめる |
| `<span>` | インライン要素 | 文中の一部分だけを装飾する |

---

## ブロック要素 vs インライン要素

| 種類 | 特徴 | 代表タグ |
|------|------|--------|
| ブロック要素 | 縦に積み重なる・幅いっぱい広がる | `<div>` `<p>` `<h1>` `<ul>` |
| インライン要素 | 横に並ぶ・テキストの一部 | `<span>` `<a>` `<img>` `<strong>` |

---

## HTML5のセマンティックタグ

意味のある場所には `<div>` の代わりに専用タグを使う：

| タグ | 用途 |
|------|------|
| `<header>` | ページやセクションのヘッダー |
| `<footer>` | ページやセクションのフッター |
| `<main>` | メインコンテンツ |
| `<nav>` | ナビゲーションメニュー |
| `<section>` | ひとまとまりのセクション |
| `<article>` | 独立した記事・コンテンツ |
| `<aside>` | サイドバー・補足情報 |

```html
<header>
  <h1>サイトタイトル</h1>
  <nav>...</nav>
</header>
<main>
  <article>...</article>
  <aside>...</aside>
</main>
<footer>...</footer>
```
